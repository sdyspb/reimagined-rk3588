# Image Signal Processor (Процессор обработки изображений)

- **Video Capture (VICAP)** (Видеозахват)
  - Поддержка **BT601**, **BT656**, **BT1120** (без перевода – стандарты)
  - Поддержка шести интерфейсов **MIPI CSI/DSI** (до 4 идентификаторов на каждый интерфейс)
  - Поддержка пяти форматов данных **CSI**: **RAW8/10/12/14**, **YUV422**
  - Поддержка трёх режимов **HDR**: **virtual channel mode** (режим виртуального канала), **identification code mode** (режим идентификационного кода), **line counter mode** (режим счётчика строк)
  - Сквозная передача **RAW** данных в **ISP0/1**

- **Maximum input** (Максимальный входной поток)
  - **48M**: 8064×6048@15 dual ISP (два ISP)
  - **32M**: 6528×4898@30 dual ISP
  - **16M**: 4672×3504@30 single ISP (один ISP)

- **3A** (Автофокус, автоэкспозиция, автобаланс белого): включает вывод статистики **AE**/**Histogram** (гистограмма), **AF**, **AWB**

- **FPN (Fixed Pattern Noise)** (Фиксированный шум узора): удаление

- **BLC (Black Level Correction)** (Коррекция уровня чёрного)

- **DPCC (Static/Dynamic defect pixel cluster correction)** (Коррекция кластеров дефектных пикселей: статических и динамических)

- **PDAF (Phase Detection Auto Focus)** (Фазовый автофокус)

- **LSC (Lens Shading Correction)** (Коррекция затенения объектива)

- **Bayer-2DNR (Spatial Bayer-raw De-noising)** (Пространственное шумоподавление в Bayer RAW)

- **Bayer-3DNR (Temporal Bayer-raw De-noising)** (Временное шумоподавление в Bayer RAW)

- **CAC (Chromatic Aberration Correction)** (Коррекция хроматических аберраций)

- **HDR (High-Dynamic Range)** (Расширенный динамический диапазон): слияние 3 кадров

- **DRC (Dynamic Range Compression)** (Компрессия динамического диапазона), **Tone mapping** (тональное отображение)

- **GIC (Green Imbalance Correction)** (Коррекция дисбаланса зелёного канала)

- **Debayer** (Демазаика): продвинутая адаптивная демазаика с коррекцией хроматических аберраций

- **CCM/CSM** (Коррекция цвета, преобразование цветового пространства): **Color Correction Matrix** (матрица коррекции цвета), **RGB2YUV** и др.

- **Gamma** (Гамма-коррекция): выходная гамма-коррекция

- **Dehaze/Enhance** (Удаление дымки/Улучшение): автоматическое удаление дымки и улучшение эффектов

- **3DLUT** (3D-таблица преобразования цветов): цветовая палитра для заказчика

- **LDCH (Lens Distortion Correction Horizontal)** (Коррекция искажений объектива только по горизонтали)

- **YUV-2DNR (Spatial YUV De-noising)** (Пространственное шумоподавление в цветовом пространстве YUV)

- **Sharp** (Резкость): повышение резкости и граничная фильтрация

- **CMSK (Privacy Mask)** (Маска конфиденциальности)

- **GAIN** (Усиление): локальное усиление изображения

- **Multi-sensor reuse ISP** (Повторное использование ISP для нескольких сенсоров)

- **FishEye Correction (FEC)** (Коррекция «рыбьего глаза»)
  - **Input mode and data format** (Режим ввода и формат данных)
    - Поддержка **RASTER**: **YUV422SP**, **YUV422I**, **YUV420SP**
  - **Output mode and data format** (Режим вывода и формат данных)
    - **RASTER**: **YUV422SP**, **YUV422I**, **YUV420SP**
    - **FBCE**: **YUV422SP**, **YUV420SP**
  - Поддержка двух плотностей: **16×8**, **32×16**
  - Коэффициент уменьшения до **4×**
  - Разрешение **128×128 ~ 4095×4095**
  - **Y Interpolation** (Интерполяция яркости): **Bicubic** (бикубическая); **C Interpolation** (интерполяция цветности): **Bilinear** (билинейная)
